## Реализованные функции (9 приоритетов)

### 1. Корпусная засвидетельствованность
Для каждой ячейки парадигмы показано **абсолютное число употреблений** в DCS.  
Источник: `CTX[form][0]` — число из конкордансного индекса.  
В списке — рядом с каждой формой в скобках: `karoti (1 450 уп.)`.

### 2. Контекстные примеры из корпуса
Клик на ячейку таблицы → панель `ctx-panel` с реальными строфами из DCS.  
Для каждой формы (P. и Ā.) показаны до 3 примеров с подсветкой формы и указанием источника.  
Примеры берутся из `conc_part1/2/3.json` (6 423 форм, до 5 строф каждая).

### 3. Разбивка основа + окончание
Кнопка **«Основа + -оконч.»** включает режим:
- основа выводится серым цветом
- личное окончание — синим с подчёркиванием
- работает для всех 6 корней и всех времён

Алгоритм: `stemSplit()` — поиск известных окончаний с конца формы (список ~30 суффиксов + редупликация для Perfect).

### 4. Сравнение двух корней
Кнопки **«Сравнить с √»** (6 вариантов).  
В каждой строке списка под основными формами появляется зелёным курсивом:  
`√bhū: bhavet / bhaveta`  
Позволяет видеть разницу классов в одной строке без переключения.

### 5. Метка класса глагола
Под переключателем корней — одна строка:  
`Класс VIII · Основа: karo-/kuru- (u-class) · P. & Ā. · основа -u- в слаб. ступени`  
Обновляется при смене корня.

| Корень | Класс | Основа | Залог |
|--------|-------|--------|-------|
| √kṛ | VIII | karo-/kuru- | P. & Ā. |
| √bhū | I | bhava- | P. & Ā. |
| √as | II | as-/s- | P. только |
| √gam | I | gaccha- | P. только |
| √vac | II | vac-/vak- | P. только |
| √dā | III | dadā-/dat- | P. & Ā. |

### 6. Маршрут обучения
Под ползунком охвата — строка:  
`▶ Следующие формы (+15%): babhūva (Perfect · 3-е sg) → kariṣyati (Future · 3-е sg) → …`  
Обновляется при смене ползунка и корня.

### 7. Карточки (flashcard)
Кнопка **⚑ Карточки** открывает колоду из форм текущего охвата.

**Вопрос:** `√kṛ · Optative · 3-е sg`  
**Ответ:** `P. kuryāt · Ā. kurvīta` + число употреблений + пример из корпуса  
**Самооценка:** ✗ Не знал / ~ Вспомнил / ✓ Знал  
**Повтор:** формы со знаком ✗ добавляются в конец колоды.  
**Навигация:** ‹ › + «Перемешать» + итог сессии.

### 8. Фильтр нулей
Кнопка **«Скрыть»** в группе «Нули» заменяет ячейки с `—` на бледную точку `·`.  
Таблица сжимается: √as в ātmanepada, √gam в du/pl Pres. Passive и т.д. перестают занимать место.

### 9. Экспорт
- **↓ CSV** — все 87 строк: Ранг, Форма, Время, Лицо, Число, P. пример, Ā. пример, Употреблений, %корпуса, ∑%
- **↓ .md** — Markdown-таблица полной парадигмы с числами употреблений + нефинитные формы

## Технические детали

- **Формат:** standalone HTML, без зависимостей от сервера
- **Размер:** ~92 KB
- **JS-синтаксис:** проверен Node.js v22 (`node --check`)

- **Concordance:** примеры из `conc_part1/2/3.json` (621 109 строф DCS 7.txt)
